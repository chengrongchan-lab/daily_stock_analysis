# AI Investment Workstation V1

加拿大 / 美股 AI 投资日报工具。

## 需要的 GitHub Secret

至少添加其中一个：

- `GEMINI_API_KEY`
- `GOOGLE_API_KEY`

你已经有 Google AI Studio Key，可以同一个 Key 同时添加成这两个名字。

## 自选股

编辑 `tickers.json`。

## 手动运行

GitHub 仓库顶部：Actions → Daily AI Investment Report → Run workflow。

运行后报告会保存到 `reports/` 文件夹。
